# scala-cli-maven-spring-failsafe-jacoco-testng-test-hello-world

## Description
A POC for spring app using testng
framework with jacoco and failsafe plugins.

## Tech stack
- scala
- maven
  - spring
  - testng
  - failsafe
  - jacoco

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
